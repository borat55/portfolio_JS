Coin Tracker

It shows the information of cryptocurrency. If a name of the cryptocurrency is clicked, the page will be shown all about the cryptocurrency by explaining about the coin, comparing the price of coin, and visualizing the price per each time or day in line chart/candle stick chart. It fetches data from cryptocurrency API by using React Query, and presents the information in real time in the page. The light and dark mode is built with Styled Components and Recoil. The charts are built with Apex Chart.

Chrome App Clone Coding

Chrome App that has a to-do list, date/time, weather forecast and quote for the day. In the to-do list, there are 3 categories; to-do, doing, done. A to-do item can be mosed to other categories from the default value, to-do, according to the status of the to-do item. For the weather forecast, if a name of city is entered correctly, it shows the weather information of the city. For the Todo list, React Hook Form is used to validate forms, and Recoil to change / manage the status of components. The weather information is pulled from Openweathermap, quote is receiving from Adviceslip.

netflex Clone Coding

A clone website of Netflix, providing a similar user experience and functionality. It allows users to browse and discover information on movies and TV shows, just like the original Netflix platform. The application utilizes data pulled from the Movie Database (TMDB) API to populate its content.
- Information on movies/tv series pulled from Movie Database(TMDB) API
- Users can easily search for specific movies or TV shows and explore various categories such as Now Playing, Popular, Top Rated
- Incorporated engaging animations and interactive features like sliders and modals using Framer Motion
